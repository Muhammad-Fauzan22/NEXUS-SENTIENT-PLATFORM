# --- CATATAN PENTING ---
# 1. Instalasi:
#    Jalankan perintah berikut di terminal Anda:
#    pip install -r requirements.txt
#
# 2. Virtual Environment:
#    Sangat disarankan untuk menggunakan virtual environment Python.
#    Buat: python -m venv .venv
#    Aktifkan (Windows): .venv\Scripts\activate
#    Aktifkan (Mac/Linux): source .venv/bin/activate
#    Lalu jalankan pip install -r requirements.txt
#
# 3. Dependensi Sistem:
#    - `weasyprint`: Mungkin memerlukan instalasi dependensi sistem seperti GTK, Cairo, Pango.
#      Lihat dokumentasi resmi WeasyPrint untuk instruksi instalasi.
#    - `pdfkit`: Memerlukan instalasi `wkhtmltopdf` di sistem Anda secara terpisah.
#      Unduh dari: https://wkhtmltopdf.org/downloads.html
#
# 4. Versi:
#    Versi yang ditentukan menggunakan >= dan < untuk memastikan kompatibilitas
#    sambil memungkinkan patch releases. Anda bisa menyematkan versi spesifik
#    (misalnya, openai==1.59.0) jika diperlukan untuk stabilitas.
